# OsmDotRoute

**.NET ネイティブの OSM 経路計算ライブラリ — 動的制約対応**

## プロジェクトの位置づけ

`災害廃棄物処理シミュレーション` プロジェクトから派生したスピンオフ。
Itinero 1.x（メンテナンス停止）からの脱却と、災害・工事・イベント時の動的通行制限に対応した経路計算機能を目的とする。

## 現在のフェーズ

**要件定義フェーズ**（Phase 0）

実装は開始していません。最初の AI Agent セッションでは `CLAUDE.md` の指示に従い、要件定義書の作成から始めること。

## 段階的開発計画

| Phase | 目標 | 状態 |
|-------|------|------|
| Phase 0 | 要件定義 | **進行中** |
| Phase 1 | 経路探索エンジン独自化（Itinero をデータ層として残す） | 未着手 |
| Phase 2 | 中間グラフフォーマット策定、ランタイム Itinero 依存削除 | 未着手 |
| Phase 3 | OSM PBF パーサー独自化（完全独立） | 未着手 |
